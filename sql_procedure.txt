        4. Quando se deve usar uma view ou um procedimento armazenado. Quais suas diferenças?

Uma View, como o próprio nome diz, é uma visualização. São construidas, basicamente, por selects. Já a Stored Procedure é uma sequência de procedimentos. São construidas por selects, updates, inserts, create table etc. A procedure não necessáriamente retorna dados e seu uso é bem mais amplo do que de uma View.
